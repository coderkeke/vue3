# Project Rules

- 不使用 any (优先使用 unknown)
- Vue Template 属性顺序符合 vue/attributes-order 规范 (DEFINITION -> LIST -> CONDITIONALS -> RENDER -> GLOBAL -> UNIQUE -> SLOT -> TWO_WAY -> DIRECTIVES -> ATTRS -> EVENTS -> CONTENT)
- Vue Template 属性名必须使用 kebab-case (vue/attribute-hyphenation)
- 关键代码必须有中文注释
- 使用less 编写样式
